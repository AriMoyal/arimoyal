## What I do

I focus on turning noisy security and infrastructure data into useful workflows, reports, alerts, and tooling.

- Build blue-team automation for log analysis, alert enrichment, and incident response
- Create practical scripts for Windows, Linux, Active Directory, endpoint visibility, and reporting
- Automate analyst workflows using n8n, Python, PowerShell, APIs, and security platforms
- Work with vulnerability intelligence, OSINT, Cloudflare logs, Wazuh, Velociraptor, and FortiEMS
- Design small security tools that solve real operational problems instead of becoming another dashboard nobody opens

## How I think about tooling

> The best security tool is not the one with the loudest dashboard.  
> It is the one that reduces noise, explains risk clearly, and helps someone take the next action without needing three meetings and a sacrificial spreadsheet.
