## Security Automation Engineer  
### Blue-Team Toolmaker · Systems Administrator · Workflow Builder

</div>

I build practical security and infrastructure automation that helps teams move faster, reduce noise, and respond with better context.

My work sits where **security operations**, **systems administration**, **automation**, and **incident response** collide. Naturally, that collision usually produces logs, tickets, alerts, and other artifacts of human suffering. I try to make them useful.

---
